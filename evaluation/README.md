# FarmGemma Evaluation Framework

## Overview
Evaluation metrics and test datasets for FarmGemma model.

## Metrics

### 1. Agricultural Accuracy
- Disease identification accuracy
- Pest detection precision/recall
- Treatment recommendation accuracy

### 2. Language Quality
- BLEU score for translations
- ROUGE score for summaries
- Fluency assessment for generated text

### 3. Multimodal Performance
- Image classification accuracy
- Vision-language alignment scores

### 4. Farmer Usability
- Response relevance score (1-5)
- Helpfulness rating
- Comprehension score

Location: `evaluation/test_data/crop_diseases/`
